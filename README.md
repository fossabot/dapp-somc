Service Offering Messaging Channel
======
"SOMC" required for messaging between client and agent before they known IP of each other.

## Dependencies

* nodejs v9.3+
* npm v5.6+
* [mongodb](https://docs.mongodb.com/manual/installation/)

## Installation

    $ mkdir SOMC && cd SOMC
    $ git clone [repo url]  
    $ npm install 

## Testing
To run the test programs, run:

    $ npm test
